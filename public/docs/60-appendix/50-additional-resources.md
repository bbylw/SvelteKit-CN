## Examples

我们编写并发布了一些不同的 SvelteKit 站点作为示例：

- [`sveltejs/realworld`](https://github.com/sveltejs/realworld) 包含一个示例博客站点
- [一个 HackerNews 克隆](https://github.com/sveltejs/sites/tree/master/sites/hn.svelte.dev)
- [`svelte.dev`](https://github.com/sveltejs/svelte.dev)

SvelteKit 用户也在 GitHub 上发布了大量示例，位于 [#sveltekit](https://github.com/topics/sveltekit) 和 [#sveltekit-template](https://github.com/topics/sveltekit-template) 主题下，以及 [Svelte Society 站点](https://sveltesociety.dev/recipe/sveltekit-templates-and-examples-e789ed397e7f38fc) 上。注意，这些尚未经过维护者的审查，可能并非最新。

## Support

你可以在 [Discord](/chat) 和 [StackOverflow](https://stackoverflow.com/questions/tagged/sveltekit) 上寻求帮助。请先先在 FAQ、Google 或其他搜索引擎、问题跟踪器以及 Discord 聊天记录中搜索与你问题相关的信息，以尊重他人的时间。提问的人远比回答的人多，因此这将有助于社区以可扩展的方式成长。
